## Descriere Proiect
Acest proiect vizează dezvoltarea unei soluții pentru monitorizarea consumului energetic folosind o arhitectură modernă. Datele sunt simulate în timp real, stocate într-o bază de date PostgreSQL și vizualizate prin dashboard-uri interactive în Grafana.
